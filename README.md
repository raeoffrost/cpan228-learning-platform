# CPAN-228 Online Learning Platform

## Student
Abby Friesen

## Project Description
This project is an Online Learning Platform built using Spring Boot. 
It allows instructors to create courses, students to enroll, submit assignments, and receive grades.

The system will implement:
- Role-based authentication (Admin, Instructor, Student)
- Course management
- Enrollment tracking
- Assignment submissions
- Grade management

## Technologies
- Spring Boot
- Thymeleaf
- Spring Data JPA
- Spring Security
- H2 Database
- Maven

## How to Run
1. Clone the repository
2. Run: ./mvnw spring-boot:run
3. Visit: http://localhost:8080
